## 第1课：大学生公司注册指南

**时长**: 约90分钟 | **类型**: 认知+实操

### 核心概念

公司注册是OPC从"个人副业"走向"正规经营"的第一步。2026年，中国公司注册流程已大幅简化，大部分城市支持"一网通办"，最快3个工作日完成注册。但选择什么类型的公司，直接影响你的税负、风险承担和未来发展。

::: warning 2026年关键政策变化
1. **新《公司法》2024年7月实施，2026年已全面落地**：有限责任公司注册资本须在5年内实缴（之前是认缴制无期限），这对OPC创业者影响很大——不要随意填"100万"注册资本。
2. **个体工商户2026年税收优惠延续**：月销售额10万元以下（含）的增值税小规模纳税人，继续免征增值税；年应纳税所得额不超过200万元的部分，减半征收个人所得税。
3. **全国"一网通办"覆盖率已达98%**：绝大多数城市公司注册全程电子化，不用跑线下窗口。
:::

### 公司类型对比：个体户 vs 有限责任公司 vs 个人独资企业

| 对比维度 | 个体工商户 | 有限责任公司（一人有限公司） | 个人独资企业 |
|---------|-----------|--------------------------|------------|
| **法律地位** | 自然人，不是法人 | 独立法人 | 非法人 |
| **责任承担** | 无限责任（个人财产连带） | 有限责任（以出资额为限） | 无限责任 |
| **注册资本** | 无要求 | 最低1元，但5年内须实缴 | 无要求 |
| **税种** | 个人所得税（经营所得） | 企业所得税（25%）+ 分红个税（20%） | 个人所得税（经营所得） |
| **税收优惠** | 月10万以下免增值税，减半个税 | 小微企业：年利润≤300万，实际税负5% | 类似个体户 |
| **注册难度** | ★☆☆☆☆ 最低 | ★★★☆☆ | ★★☆☆☆ |
| **做账报税** | 简单（可自行报税） | 复杂（需专业记账） | 中等 |
| **开发票** | 可开票，额度有限 | 正常开票 | 正常开票 |
| **接大客户** | 受限（大公司不认个体户） | 不受限 | 不受限 |
| **社保公积金** | 可自主缴纳 | 必须缴纳 | 可自主缴纳 |
| **转让/融资** | 不可转让 | 可转让股权 | 可转让 |
| **注册费用** | 0-500元 | 0-1000元（含刻章） | 0-500元 |
| **适合场景** | 小额服务、兼职副业 | 有一定规模、想做大 | 风险较低的专业服务 |

::: tip OPC创业者的选择建议
- **月收入＜1万，刚起步** → 先不注册，以个人身份接单，用"自然人代开发票"
- **月收入1-3万，稳定接单** → 注册个体工商户，税负最低，够用
- **月收入3万+，想做大，想融资** → 注册一人有限责任公司
- **有风险的专业服务（如法律咨询、财务顾问）** → 有限责任公司，隔离个人风险
:::

### 公司注册全流程（2026年简化版）

```
第1步：核名（1-2天）
  → 在"当地市场监督管理局"官网或"一网通办"平台提交2-3个备选名称
  → AI工具：用DeepSeek帮你生成好记的公司名

第2步：网上设立登记（1-3天）
  → 提交：法人身份信息、注册地址、经营范围、注册资本
  → 2026年大部分城市支持"住所承诺制"（大学生可用学校地址或孵化器地址）

第3步：领取营业执照（0-1天）
  → 电子营业执照自动发放，纸质版可邮寄

第4步：刻章（0-1天）
  → 公章、财务章、法人章、发票章
  → 很多城市为新注册公司免费刻章（"政府买单刻章"）

第5步：银行开户（1-3天，需线下）
  → 预约银行开对公账户
  → 推荐：招商银行、工商银行（对中小企业友好）

第6步：税务登记（0-1天）
  → 电子税务局自动完成，无需单独跑
  → 核定税种、领取发票

第7步：社保/公积金开户（可选）
  → 如果只有你自己，可以选择不交社保（保留学生身份）
  → 如果全职创业，需要在公司名下缴纳社保
```

## 第2课：AI法律辅助

**时长**: 约90分钟 | **类型**: 认知+实操

### 核心概念

OPC创业者的法律需求集中在三个场景：**合同审查**（我签的合同有没有坑？）、**合同生成**（我需要一份合同模板）、**知识产权保护**（我的品牌/产品怎么保护？）。AI不能替代律师，但可以帮你做第一轮审查，标记风险条款，节省80%的法律咨询费用。

::: warning 重要声明
AI法律辅助工具不能替代专业律师。涉及重大金额、重要合作、法律纠纷时，务必咨询持证律师。本课内容仅供学习和初步参考。
:::

### 动手实操1：用AI审查合同（30分钟）

#### Step 1：合同审查Prompt模板

```text
你是一位经验丰富的合同法律师。请帮我审查以下合同，重点关注对我不利的条款。

[粘贴合同全文]

请从以下维度审查：

1. **风险条款**（标注为🔴高风险 / 🟡中风险 / 🟢低风险）：
   - 违约责任条款：是否对我不公平？违约赔偿是否过高？
   - 付款条款：付款条件、时间、方式是否合理？
   - 知识产权条款：我的知识产权是否会流失？
   - 保密条款：是否合理保护双方商业秘密？
   - 终止/解约条款：我能否单方面终止？后果是什么？
   - 争议解决条款：管辖法院/仲裁在哪个城市？（异地诉讼成本高）
   - 竞业限制条款：是否限制我未来的职业发展？

2. **缺失条款**：
   - 合同中缺少哪些对我有利的条款？
   - 建议增加的条款（如：验收标准、交付物定义、售后服务范围）

3. **修改建议**：
   - 每个风险条款的具体修改建议
   - 修改后的条款文本（标注修改部分）

4. **总体评价**：
   - 这份合同对我是"可以签"、"需要修改再签"还是"建议拒绝"？
   - 如果必须签，哪些条款需要重点谈判？

请用中文输出，法律术语用通俗语言解释。
```

#### Step 2：常见合同类型专项审查

**服务合同审查重点**：

```text
请针对以下服务合同，重点审查：

[粘贴合同]

额外关注：
1. 服务范围是否清晰定义？（避免"范围蔓延"）
2. 交付物和验收标准是否明确？
3. 修改/返工条款：多少次修改是免费的？
4. 付款节点：是否与交付物挂钩？
5. 知识产权归属：交付物版权归谁？
6. 客户不配合/拖延怎么办？（如客户不提供素材、不反馈）
```

**合作协议审查重点**：

```text
请针对以下合作协议，重点审查：

[粘贴合同]

额外关注：
1. 双方的投入和产出是否对等？
2. 利润分配机制是否清晰？
3. 决策机制：谁说了算？僵局怎么解决？
4. 退出机制：如果合作不下去了，怎么分家？
5. 排他条款：是否限制我接其他合作？
```

### 动手实操3：知识产权基础（25分钟）

### 税务风险避坑指南（15分钟）

::: danger OPC创业者最常见的5个税务坑

**坑1：以为没收入就不用报税**
- 真相：即使零收入，也要按时进行"零申报"，否则会被列入经营异常名录
- 后果：影响个人征信，贷款、买房、出国可能受限

**坑2：公私不分，个人账户收公司款项**
- 真相：公司收入必须进对公账户，不能直接转到个人微信/支付宝
- 后果：税务稽查时可能被认定为偷税漏税

**坑3：不开发票就不报这笔收入**
- 真相：无论是否开发票，所有收入都应纳税
- 后果：金税四期系统2026年已全面运行，银行流水与税务申报自动比对，隐瞒收入极易被查出

**坑4：随意填注册资本**
- 真相：新《公司法》规定注册资本须在5年内实缴。填100万，5年内要拿出100万。
- 建议：OPC创业者填1-10万即可

**坑5：不记账或乱记账**
- 真相：即使是个体工商户，也需要记账。凭证不全，税务稽查时无法自证清白
- 建议：用AI记账工具，每月花30分钟，养成习惯
:::

## FAQ

### Q1：我是大学生，现在注册公司会影响我的应届生身份吗？
**A**：这是很多同学最关心的问题。分情况讨论：
- **在校生注册个体工商户**：不影响应届生身份，因为你仍然是"在校学生"。
- **在校生注册有限责任公司（担任法人）**：一般不影响应届生身份，但如果你在公司缴纳了社保，部分企业会认为你"已有工作经历"，可能影响校园招聘。建议：在校期间注册个体户，不缴社保，保留应届生身份。
- **毕业后注册公司**：你已经是"往届生"，不存在影响应届生身份的问题。

### Q2：我没有钱注册公司，怎么办？
**A**：2026年注册公司的成本已经非常低。个体工商户注册0-200元，很多城市还免费刻章。有限责任公司注册0-500元。如果你连几百块都没有，可以先不注册——以个人身份接单，用个人收款码收款，等月收入稳定在3000元以上再注册。

### Q3：AI审查合同靠谱吗？会不会漏掉重要风险？
**A**：AI审查合同可以做"第一轮筛选"，但**不能替代律师**。建议：合同金额＜5000元 → AI审查即可；合同金额5000-20000元 → AI审查 + 自己仔细阅读；合同金额＞20000元 → 找专业律师审查。AI审查的价值在于：标记出80%的常见风险条款，让你知道"哪里需要重点关注"。

### Q4：金税四期是什么？对我有什么影响？
**A**：金税四期是中国税务系统的第四代升级，2025年全面上线，2026年已稳定运行。核心变化是：税务系统与银行系统联网，能自动比对"你的银行流水"和"你的报税申报"。对OPC创业者的影响：**不要试图隐瞒收入**，所有收入（无论是否开发票）都应如实申报。合法避税可以（如利用小微企业税收优惠），偷税漏税不行。

### Q5：我想做海外市场，怎么收外币？
**A**：如果是海外个人客户，用PayPal最简单（个人即可注册，但费率较高）。如果是海外企业客户，用Stripe（需要海外公司资质）或Wise（支持多币种账户）。如果是小额打赏/赞助，用Buy Me a Coffee或Patreon。2026年，支付宝和微信支付也支持部分海外地区的收款，但覆盖面和费率不如PayPal/Stripe。

### Q6：5个案例都月入过万了，我感觉自己差很远，怎么办？
**A**：不要被"月入过万"吓到。这5个案例的平均时间跨度是12个月，每个都是从0开始的。关键不是"他们现在赚多少"，而是"他们的第1步做了什么"。小林的第1步是"在闲鱼发了一条50元的翻译服务"，小王的第1步是"免费帮朋友的咖啡店做账号"。你的第1步不需要多大，先赚到第1块钱，后面的路自然就清晰了。
