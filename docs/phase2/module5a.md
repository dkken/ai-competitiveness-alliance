## 第1课：AI简历优化

**时长**: 约90分钟 | **类型**: 实操为主

### 核心概念

#### 简历的本质是什么？

在AI时代的招聘流程中，你的简历首先要通过**两层筛选**：

```
第一关：ATS机器筛选 → 关键词匹配、格式解析
第二关：HR人工筛选 → 6秒初筛 → 30秒精读
```

简历的核心不是"你做过什么"，而是 **"你能解决什么问题"**。每个经历描述都应该回答三个问题：

1. 你面对什么问题/挑战？
2. 你做了什么来解决问题？
3. 结果如何？（最好有量化数据）

### 动手实操：AI全面优化简历

#### Step 1：JD关键词提取与能力模型（15分钟）

打开任意一个AI工具（推荐DeepSeek，中文理解最好），将你目标岗位的JD粘贴进去：

```text
你是一位有10年经验的HR总监。请帮我分析以下岗位JD：

[在此粘贴完整的岗位JD]

请完成以下分析：

1. **核心关键词清单**：提取JD中最关键的15-20个关键词（技能、工具、素质），按重要性排序

2. **隐性能力需求**：JD里没有明说但是面试官实际在考察的3-5项能力

3. **候选人画像**：用一段话描述这个岗位的"理想候选人"长什么样

4. **ATS筛选逻辑推断**：如果你是ATS系统，你会设置哪些关键词作为简历筛选的"通过条件"？

5. **简历匹配建议**：基于以上分析，我的简历最应该突出哪3个方面的经历？
```

**实操记录**：将AI输出的关键词清单保存下来，这是你接下来修改简历的"标准答案"。

#### Step 2：简历逐模块AI优化（35分钟）

**原则**：分段优化，不要一次性提交整份简历。逐模块精修。

**2A. 个人总结/自我评价优化**

将你现有的个人总结粘贴进AI：

```text
你是一位简历优化专家。以下是我的个人总结部分，请帮我优化：

[粘贴你现有的个人总结]

优化要求：
1. 在3-4句话内说明：专业背景 + 核心能力 + 求职方向 + 独特价值
2. 自然融入以下JD关键词：[从Step 1中选取5-8个最重要的关键词]
3. 避免空话套话（如"本人性格开朗、吃苦耐劳"），每句话都要有信息量
4. 语气专业但不死板，展现自信但不浮夸
5. 给我3个版本：精炼版（50字）、标准版（80字）、详细版（120字）
```

**2B. 工作/实习经历优化 —— STAR法改写**

这是简历最有分量的部分。选择你的一段经历：

```text
你是一位简历优化顾问。请用STAR法则帮我改写以下经历描述：

STAR法则 = Situation（情境）→ Task（任务）→ Action（行动）→ Result（结果）

我的原始经历描述：
[粘贴你的一段经历，如"在大二暑假去XX公司实习，做市场助理，负责公众号运营……"]

请帮我改写为STAR格式：
- S（情境）：用一句话交代背景
- T（任务）：用一句话说清你要做什么
- A（行动）：用2-4个动词开头的短语，描述你具体做了什么
- R（结果）：量化你的贡献——数据、百分比、排名、表彰等

附加要求：
1. 每个行动描述用动词开头（如"搭建"、"主导"、"提升"、"优化"）
2. 结果必须有量化数据。如果确实没有精确数据，用合理的估计（标注"约"）
3. 控制总字数在80-120字
4. 自然融入JD关键词
5. 同时给我一个"A/B版本"——A版更量化、B版更叙事，你自己选
```

**常用STAR动词库**（收藏备用）：

| 类别 | 动词 |
|------|------|
| 领导/管理 | 主导、统筹、带领、协调、推动、发起 |
| 分析/研究 | 分析、调研、评估、诊断、洞察、预测 |
| 创造/建设 | 搭建、创建、设计、策划、研发、打造 |
| 提升/优化 | 提升、优化、改进、精简、加速、突破 |
| 执行/落地 | 执行、落地、交付、完成、推进、落实 |

**2C. 项目经历/校园经历优化**

```text
你是一位校园招聘专家。请帮我优化以下项目/校园经历：

[粘贴你的项目或校园经历]

请从以下角度改写：
1. 突出你"解决了什么问题"而非"参加了一个什么活动"
2. 强调可迁移能力（沟通协调、数据处理、内容创作、项目管理）
3. 如果经历比较普通，请帮我找到"不普通的表达角度"
4. 控制在80-100字
```

**2D. 技能与证书部分优化**

```text
请基于以下JD的关键需求，帮我优化技能和证书部分的表述：

目标岗位：[岗位名称]
JD核心关键词：[从Step 1来的]

我的技能清单：[列出你真实的技能，如"Word、Excel、PPT、PS、英语六级"]

优化要求：
1. 将技能按"核心技能"和"辅助技能"分类
2. 不要简单罗列工具名，用"能做什么"来表达（如不说"精通Excel"，而说"能用Excel进行数据透视和可视化分析"）
3. 标注技能的熟练程度（熟练/掌握/了解）
4. 证书按含金量排序，含金量不高的放在后面或不写
```

#### Step 3：AI简历评分与诊断（20分钟）

整份简历优化完成后，让AI做最后的全面检查：

```text
你是一位严苛的HR总监。以下是我的优化后简历全文，请帮我做一个全面的质量评估：

[粘贴完整简历]

请从以下6个维度打分（各1-10分）并给出改进建议：

1. **ATS友好度**：关键词覆盖度、格式是否会损坏、推荐系统识别度
2. **第一印象力**：6秒内能否让HR产生兴趣、核心信息是否突出
3. **STAR法则应用**：经历描述是否有情境-任务-行动-结果
4. **量化程度**：经历的成果是否有数据支撑
5. **无冗余度**：是否每一句话都有存在的必要、有没有废话套话
6. **岗位匹配度**：整体与目标岗位的契合度

最后，列出TOP 3最亟需改进的问题，并给出具体的改写建议（不要只是批评，要给出改写示例）。
```

#### Step 4：多版本简历生成（20分钟）

不同公司需要不同侧重点的简历：

```text
基于我刚才优化的简历，请帮我生成3个不同版本的简历个人总结和经历描述：

版本1 - **互联网大厂版**：强调数据思维、结果导向、快速迭代、项目执行力
版本2 - **国企/央企版**：强调综合素质、团队协作、稳定性、政治面貌（如有）、在校荣誉
版本3 - **外企/跨国公司版**：中英文双语格式、强调跨文化沟通、国际化视野、自学能力

每个版本仅优化"个人总结"和"第一段经历描述"。
```

### 避坑指南

| ❌ 常见错误 | ✅ 正确做法 |
|-----------|-----------|
| 让AI编造不存在的经历 | AI只能优化你真实做过的事，编造经历是诚信问题 |
| 一份简历投所有岗位 | 每个JD都需要针对性调整关键词和侧重点 |
| AI优化后用花哨模板 | 很多ATS系统无法解析两栏/图表/特殊字体，用简洁单栏排版 |
| 完全复制AI的输出 | AI是你的"编辑"不是你的"替身"，最终内容必须自己审核 |
| 忽略格式细节 | 检查PDF导出后的字体嵌入、特殊字符是否正常显示 |

## 第2课：AI模拟面试

**时长**: 约90分钟 | **类型**: 实操为主

### 核心概念

**面试不是"考试"，是"对话"**。最好的面试状态是：你在向一个对你感兴趣的同行解释你的工作，而不是在回答HR的"拷问"。

面试本质是三个问题：
1. 你能胜任这份工作吗？（能力）
2. 你愿意做这份工作吗？（动机）
3. 我们愿意和你共事吗？（契合度）

AI模拟面试的优势是你可以在零压力环境下无限次练习，直到形成肌肉记忆。

### 动手实操：AI立体面试训练

#### Round 1：STAR法则强化训练（30分钟）

**什么是真正的STAR？**

面试官想通过STAR看到的不是"你做了A、B、C"，而是"你为什么选A而不是B，你怎么克服C中的困难，结果D到底有多大"。

```text
你现在是一位[目标公司类型，如：字节跳动/腾讯]的[目标岗位，如：产品经理]面试官。

请基于我的简历内容，向我提3个STAR类行为面试题。

面试要求：
1. 每个问题都要逼我给出"具体情境中的具体行动"
2. 如果我回答得不够具体，请你追问："能不能再具体一点？比如……”
3. 3个问题分别覆盖：领导力、解决问题能力、失败与复盘
4. 每次我回答后，请你：
   a) 评价我的回答质量（1-10分）
   b) 指出我遗漏了STAR的哪个环节
   c) 给我一个更好的回答示例

我的简历核心经历：
[粘贴你的简历要点摘要，200字左右]

请开始第一个问题。
```

::: tip 训练建议
和AI进行至少3轮完整的STAR对话。录下你的回答，自己回听——你会发现很多自己没意识到的口头禅和逻辑漏洞。
:::

#### Round 2：技术能力面模拟（20分钟）

::: details 非技术岗同学请展开
即使你不是程序员，你的岗位也有"专业能力面试"。把下面的Prompt中的技术问题替换为你专业的核心问题即可。例如：
- 市场营销："请分析一个新消费品牌的上市策略"
- 金融："请对这家公司做一个简单的估值"
- 教育："设计一堂30分钟的互动课"
:::

```text
你现在是一位[公司名]的[岗位名]技术面试官。请对我进行一轮技术面试。

技术方向：[你的专业/技术方向，如：数据分析、前端开发……]
岗位级别：校招/初级

面试流程：
1. 先问2个基础概念题（考察基础知识扎实度）
2. 再问1个场景应用题（考察解决问题的能力）
3. 最后问1个开放思考题（考察技术视野和学习能力）

我回答后，请给出：
a) 回答中的亮点
b) 回答中的不足
c) 面试官可能会追问什么
d) 一个参考回答——这是你希望听到的满分答案

我的技术背景简述：
[简要描述你的技术水平，如：熟悉Python数据分析，做过2个课程项目……]
```

#### Round 3：不同公司类型面试模拟（25分钟）

不同公司的面试风格天差地别：

```text
你将扮演[公司类型]的面试官，对我进行一场15分钟的模拟面试。

请完全按照该类型公司的真实面试风格来：

—— 选择一种类型开始 ——

**类型A：互联网大厂（字节/腾讯/阿里/拼多多）**
风格：快节奏、压力面成分、追问细节、看重逻辑和数据
典型问题：
- "你做过的项目中，最困难的一个决策是什么？"
- "如果让你重新做这个项目，你会改变什么？"
- "你对加班怎么看？"

**类型B：国企/央企（中石油/国家电网/中国移动）**
风格：看重综合素质、政治素养、稳定性、在校表现
典型问题：
- "你为什么选择我们单位？"
- "谈谈你的优缺点"
- "你对未来的职业规划是什么？"
- "你是否是党员/预备党员？"

**类型C：外企（宝洁/联合利华/四大/咨询）**
风格：结构化面试、看重逻辑表达、英语能力、领导力案例
典型问题：
- "Tell me about a time you led a team to achieve a goal."（英文）
- "你如何处理和团队成员的分歧？"
- "为什么选择咨询/快消行业？"

请从你选择的类型开始问我第一个问题。每次我回答后，给我：
- 你的"面试官评分"（1-10分）
- 我的回答是否符合该类型公司的"潜规则"
- 如果是真实面试，我的表现处于什么水平
```

#### Round 4：薪资谈判模拟（15分钟）

::: danger 重要提醒
薪资谈判是面试中最容易被"坑"的环节。AI模拟能帮你提前准备好话术，避免现场慌乱。
:::

```text
你现在是一位HR，我们正在进行最终的薪资谈判。

背景设定：
- 公司和岗位：[填写]
- 岗位薪资范围：月薪[填写] - [填写]
- 我的期望薪资：月薪[填写]
- 目前我手上还有[填写]个其他Offer（真实与否你自己知道，这是模拟）

谈判流程（4轮）：
第1轮：你先报出offer的薪资数字，低于我的期望
第2轮：我尝试争取，你提出一些"公司政策限制"
第3轮：我用其他offer作为谈判筹码，你的反应
第4轮：达成最终数字（或僵局）

每一轮对话后，请给我分析：
- 我的谈判策略是否正确
- 还有哪些我没有利用的谈判筹码
- HR这句话的心理战术是什么
```

## 第3课：AI求职信息战

**时长**: 约90分钟 | **类型**: 认知 + 实操

### 核心概念

求职不只是投简历——它是一场信息战。你需要了解：

1. **行业格局**：这个行业在增长还是萎缩？
2. **公司状况**：目标公司财务健康吗？最近有什么大动作？
3. **竞争态势**：你的竞争对手是谁？差异化在哪里？
4. **个人品牌**：招聘平台上的你是"专业选手"还是"路人甲"？

AI能让这些调研从"一周"缩短到"一小时"。

### 动手实操：AI驱动求职情报系统

#### Task 1：行业速览分析（20分钟，推荐工具：Kimi联网搜索）

```text
我需要深入了解[目标行业，如：新能源汽车/跨境电商/人工智能应用]行业的2026年现状。

请帮我完成一份"求职者导向"的行业速览：

1. **行业规模与增长**：2025-2026年市场规模、增速（用具体数据）
2. **头部玩家**：列出Top5公司及其核心业务、2026年最新动态
3. **人才需求趋势**：最紧缺的3类岗位、平均薪资范围、入行门槛
4. **风险预警**：这个行业目前最大的3个风险（政策、竞争、技术）
5. **入行建议**：对一个应届生来说，进入这个行业的最佳切入点是什么？
6. **行业黑话**：列出8-10个这个行业的常用术语或缩写

请标注所有数据的来源。如果没有确切数据，标注"[推测]"，不要编造。
```

#### Task 2：目标公司深度分析（25分钟，推荐工具：Kimi或Claude联网搜索）

```text
我需要深度分析[目标公司名称]作为求职目标。

请帮我完成以下分析（请使用联网搜索获取最新信息）：

1. **基本面**：
   - 公司全称、成立时间、总部、员工规模
   - 核心业务线和收入构成
   - 2025-2026年最新财报亮点（营收、利润、增长率）
   - 融资/上市情况

2. **公司动态**：
   - 2026年以来最重要的3条公司新闻
   - 最近正在发力什么新业务/新产品？
   - 裁员/扩张动态

3. **雇主评价**：
   - 在脉脉/看准网/Glassdoor上，员工普遍提到的优点和槽点
```

## 第4课：求职全流程实战

**时长**: 约90分钟 | **类型**: 实操 + 整合

### 核心概念

前面三课都是"武器锻造"，这一课是"上战场"。你需要把碎片化的技能串成一个完整的求职流程，从看到JD到拿到Offer。

### 求职全流程时间线

```
Day 0:  发现目标岗位 → AI分析JD → 1小时内完成简历定制
Day 1:  AI生成求职信 → 投递简历
Day 3:  收到面试通知 → AI分析公司 → 准备面试话术
Day 5:  AI模拟面试3轮 → 准备反问问题
Day 7:  正式面试
Day 8:  AI辅助写感谢信 → 复盘面试表现
Day 14: 收到Offer → AI辅助薪资谈判
```

### 动手实操：全流程通关

#### 关卡1：真实岗位全流程模拟（40分钟）

**选择你的实战岗位**：打开BOSS直聘/猎聘/实习僧/牛客网，找一个你真正感兴趣的岗位。复制完整的JD。

然后按以下SOP执行：

```
求职SOP（标准化操作流程）

Step A | 5分钟 | JD分析
→ 将JD粘贴给AI，执行第1课Step 1的Prompt
→ 产出：关键词清单 + 候选人画像

Step B | 15分钟 | 简历定制
→ 基于关键词清单，用第1课Step 2的方法优化简历
→ 重点调整个别经历描述的侧重点，而非重写整个简历
→ 产出：针对该岗位定制的简历

Step C | 10分钟 | 求职信生成
→ 用第3课Task 4的Prompt生成求职信
→ 选择最适合该公司风格的版本
→ 微调语气，替换AI写的"你的产品真棒"之类的假热情

Step D | 10分钟 | 面试准备包
→ 用第2课Round 1和Round 2的方法生成可能的面试题
→ 针对每道题准备好STAR回答大纲（不是逐字稿！记要点即可）
→ 准备3个"反问面试官的问题"
```

#### 关卡2：反问面试官的艺术（15分钟）

面试最后的"你有什么问题要问我吗？"不是客套，是你的加分/扣分环节。

```text
你是一位面试教练。我即将面试[公司名]的[岗位名]。

请帮我准备5组"高质量反问问题"，每组针对不同的面试官角色：

1. **问HR**（1-2题）：关于团队文化、培训体系、晋升路径
2. **问直属上级**（1-2题）：关于团队现状、期望、工作方式
3. **问高管/总监**（1题）：关于业务方向、行业判断、战略思考

要求：
- 不要问官网能查到的信息（显得你没做功课）
- 不要问"公司福利怎么样"（显得你只关心待遇）
- 每个问题要展现你的思考深度和对这个岗位的热情
- 给我每个问题背后的"心机"——面试官从这个问题中能看出你的什么品质？
```

#### 关卡3：面试后跟进与复盘（20分钟）

面试结束不是终点。专业的跟进能让你的Offer率提升30%。

```text
你是一位求职策略顾问。我刚刚完成了[公司名+岗位]的面试，请帮我：

第一部分：面试感谢信（24小时内发送）
- 生成3个版本：简短版（150字）、标准版（250字）、深度版（350字）
- 每个版本必须提到面试中讨论过的1个具体话题（不是泛泛的"感谢您的时间"）
- 语气：自信但不傲慢，热情但不卑微

第二部分：面试复盘分析
请根据我提供的面试录音文字稿/面试回忆，帮我分析：
[粘贴你的面试回忆或录音文字稿]

复盘维度：
1. 我回答最好的2个问题（为什么好？）
2. 我回答最差的2个问题（差在哪？当时应该怎么回答？）
3. 面试官表现出兴趣的3个时刻（微表情/追问/点头）
4. 面试官表现冷淡的2个时刻
5. 基于以上，我的通过概率预估（%）
6. 如果没通过，最大的原因可能是什么？
```

::: warning 求职焦虑是正常的
根据Boss直聘2026年数据，应届生平均投递47份简历才能拿到1个offer。被拒不是你的问题，是概率问题。

```text
你是一位职业规划师兼心理顾问。我现在正处在求职季，感觉：

- 投了[数字]份简历，只有[数字]个面试
- 面了[数字]家，还没有拿到Offer
- 最让我焦虑的是：[具体焦虑点]

请给我：

2. 3个可以立即执行的行动建议（不是鸡汤，是具体动作）
3. 重新构建我的求职心态——如果用一个更有力量的视角来看待"被拒"这件事
```

## FAQ

**Q1：AI会不会把我的简历优化得太"模板化"，让HR觉得千篇一律？**

不会——如果你只是让AI"帮我优化简历"，那确实会模板化。正确做法是：把AI当作你的编辑，你提供素材（真实经历），AI帮你打磨表达。每次都针对具体JD定制关键词，让每份简历都有差异点。AI负责60%的"标准化优化"（语法、结构、STAR框架），你负责40%的"个性化决策"（选哪个经历、侧重哪个能力、用什么语气）。

**Q2：我没有实习经历，简历太单薄，AI也救不了吧？**

恰恰相反。没有实习经历的情况下，**更需要AI帮你挖掘"隐藏亮点"**。把课程项目、社团活动、志愿者经历、甚至课堂大作业都当作"项目经历"来描述。AI能帮你从这些看似普通的经历中提炼出可迁移的能力。关键是：**描述你解决了什么问题，而不是你参加了什么活动**。比如"担任班级学习委员" → "为52名同学搭建学习资源共享平台，期末班级平均绩点提升0.3"。

**Q3：面试时紧张到脑子一片空白，AI模拟练习有用吗？**

非常有用——因为AI模拟的目的是建立"肌肉记忆"。当你把STAR回答练了20遍、把自我介绍说了30遍，即使紧张，你的大脑也能自动执行这些已经刻在神经回路里的回答。建议：AI模拟 → 录下来 → 自己回听 → 再练 → 再录。反复4-5次，你会发现紧张感显著下降。

**Q4：AI写的求职信HR能看出来吗？会被扣分吗？**

HR大概率能感觉到"AI味"——过于完美、过于正式、缺乏具体细节。所以你需要：1）不要直接用AI输出，先让AI生成，然后你手动加入1-2个真实的个人细节（比如"我注意到贵司最近上线的XX产品，我在大学期间恰好做过类似的课程项目"）；2）调整语气，让简历和求职信读起来像"同一个人在说话"。检查方法：把你的简历和求职信一起读一遍，如果感觉像两个人写的，就继续改。

**Q5：AI面试模拟时，AI的"评分"准吗？**

不完全准确，但有参考价值。AI评分更擅长发现"结构性缺陷"（比如你忘了说Result、逻辑跳跃），而不擅长判断"人设吸引力"（你有没有亲和力、你的气场怎么样）。建议：把AI评分当作"底线检查"——AI说好的不一定真的好，但AI说差的地方大概率真的要改。
