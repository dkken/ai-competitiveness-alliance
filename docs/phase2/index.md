## 如何选择？

| 你的情况 | 建议 |
|---|---|
| 不确定想做什么，先找份好工作 | 选就业轨 |
| 有强烈的创业冲动，想做自己的事 | 选创业轨 |
| 想创业但还没准备好，先打工积累经验 | 选就业轨，结业后自学创业轨 |
| 时间充裕，两条都想学 | 先就业轨（第5-6周）→ 再创业轨（第7-8周） |
