> **核心原则**: 选择一条路走到底。就业轨专注"求职竞争力"，创业轨专注"产品变现"。你可以在完成一条轨后，用另一条轨的内容作为补充。

## 两条轨，两种未来

第二阶段是课程的"分水岭"。经过第一阶段4周的通用能力训练，你已经掌握了AI的基本使用方法和Prompt工程能力。现在，你要根据自己的目标选择一条路。

### 就业轨 vs 创业轨：核心差异

| 对比维度 | 就业轨 | 创业轨 |
|---------|--------|--------|
| 核心目标 | 拿到更好的Offer | 做出能卖钱的产品 |
| 适合人群 | 想在秋招/春招中脱颖而出的学生 | 有创业冲动、想做副业的学生 |
| 核心能力 | AI求职全流程 + AI办公效率 + 作品展示 | 产品开发 + 营销获客 + 公司运营 |
| 典型产出 | 优化简历+个人网站+作品集+职业规划 | MVP产品+商业计划书+营销方案+运营SOP |
| 时间投入 | 约36小时（4周×9小时） | 约38小时（4周×9.5小时） |
| 后续路径 | 实习→全职→职业发展 | 副业→OPC→全职创业 |

### 两条轨的关系

```
第一阶段（通用能力）
       │
       ├──→ 就业轨（模块5A-8A）──→ 拿到Offer
       │         │
       │         └──→ 第三阶段（超级个体进阶）
       │
       └──→ 创业轨（模块5B-8B）──→ 上线产品
                 │
                 └──→ 第三阶段（超级个体进阶）
```

> 两条轨最终汇入第三阶段，因为无论是就业还是创业，自动化工作流和终身学习能力都是通用的进阶能力。

## 如何选择？

| 你的情况 | 建议 | 原因 |
|----------|------|------|
| 不确定想做什么，先找份好工作 | 选就业轨 | 就业是大多数学生的现实需求，先解决"生存"问题 |
| 有强烈的创业冲动，想做自己的事 | 选创业轨 | 创业需要趁早试错，AI降低了创业门槛，大学期间是最佳试错期 |
| 想创业但还没准备好，先打工积累经验 | 选就业轨，结业后自学创业轨 | 先通过就业进入行业，了解市场需求后再创业成功率更高 |
| 时间充裕，两条都想学 | 先就业轨（第5-6周）→ 再创业轨（第7-8周） | 交替学习可以让你同时具备求职和创业的能力，但需要每周投入12-14小时 |
| 已有实习Offer，想提升职场竞争力 | 选就业轨 | 就业轨的AI办公效率模块能让你入职即展现超预期价值 |
| 已有副业想法，想用AI加速 | 选创业轨 | 创业轨的Vibe Coding和营销模块能帮你快速把想法变成产品 |

::: tip 选择建议
如果你实在纠结，选就业轨。因为：
1. 就业轨的产出（简历、作品集、AI办公技能）在你未来创业时也完全有用
2. 创业轨需要更多的自主驱动力，如果没有明确的想法，容易半途而废
3. 就业轨学完后，你可以随时回来自学创业轨的内容
:::

## 第二阶段与其前后阶段的关系

### 承接第一阶段
第二阶段使用的AI工具和Prompt方法，全部建立在第一阶段的基础上。如果你第一阶段学得不够扎实，建议在进入第二阶段前先复习模块2（Prompt Engineering）和模块3（AI工具深度实战）。

### 衔接第三阶段
第二阶段结束后，就业轨学生有了"求职竞争力"，创业轨学生有了"产品变现能力"。但无论哪条轨，你都需要第三阶段的"自动化工作流"和"终身学习体系"来持续提升自己——因为AI时代的变化速度，要求你不断进化。
