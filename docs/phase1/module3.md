## 第1课：AI写作全流程 —— 从选题到发布

**时长**: 约120分钟 | **类型**: 实操为主

### 核心概念

AI写作不是"让AI替你写"，而是 **"你当主编，AI当写手"**。你负责选题、定调、审稿，AI负责初稿、润色、多版本适配。

**AI写作的黄金公式**：

```
你的专业判断力 + AI的生成速度 = 10倍效率的写作能力
```

### 动手实操：完成一篇1000字科普文章

按照以下六阶段工作流，用AI辅助完成一篇你本专业方向的科普文章：

#### 阶段1：选题与破题（15分钟，推荐工具：DeepSeek）

```text
你是一位[你的专业领域]的资深科普作家。我在准备写一篇面向大学生的科普文章。

请帮我：
1. 从5个不同角度提出可能的选题方向（每个用一句话描述）
2. 对每个选题进行"吸引力"和"专业深度"打分（各1-5星）
3. 针对最佳选题，列出3个核心论点

我的专业：[你的专业]
目标读者：非本专业的大学生
字数要求：1000字左右
```

#### 阶段2：资料收集与梳理（20分钟，推荐工具：Kimi）

- 打开 Kimi，开启联网搜索
- 搜索你选题相关的关键词，收集2-3篇参考资料
- 用以下 Prompt 让 Kimi 帮你提炼核心信息：

```text
请帮我梳理关于"[你的选题]"的核心信息：
1. 用3个关键数据或事实作为文章支撑
2. 提取2-3个有趣的小故事或案例
3. 列出常见的3个误解和真相

请确保信息准确，不确定的地方标注"[需核实]"。
```

#### 阶段3：大纲生成（15分钟，推荐工具：Claude 或 DeepSeek）

```text
你是一位逻辑严密的学术写作导师。基于以下素材，请帮我搭建文章大纲。

[粘贴阶段2收集的素材]

大纲要求：
1. 使用"开头钩子 → 正文3个论点 → 结尾金句"的结构
2. 开头用"设问式"或"反常识式"引入
3. 每个论点下标注需要的数据和案例
4. 全文控制在1000字左右
```

#### 阶段4：分段撰写初稿（30分钟，推荐工具：DeepSeek）

**关键原则：分段撰写，不要一次性生成全文！**

```text
请帮我撰写文章的第一部分：开头（约150字）

要求：
- 用一个令人惊讶的数据或反常识的提问开头
- 自然地引出本文主题
- 暗示读者读完会有收获

如果你觉得我的开头思路不够好，请给我3个替代方案。
```

然后依次撰写正文三个论点和结尾，每段单独给出指令。

#### 阶段5：全文润色（20分钟，推荐工具：豆包 或 Claude）

```text
以下是文章初稿。请作为一位资深编辑，帮我优化：

[粘贴初稿全文]

请分4轮优化：
第1轮：检查逻辑漏洞和论证不充分的地方，标注出来
第2轮：优化语言表达，去掉冗余和废话，让句子更紧凑
第3轮：提升文采，让文章更有可读性，但不能变浮夸
第4轮：检查是否存在事实错误或过度断言，标注"[需核实]"
```

#### 阶段6：多平台适配（20分钟，推荐工具：豆包）

同一篇文章，不同平台需要不同风格。选做你最感兴趣的1个平台：

```text
请将以下文章改编成[小红书/公众号/知乎]风格：

[粘贴你的终稿]

小红书风格要求：活泼轻松、带emoji、分行短句、加话题标签
公众号风格要求：标题吸引人、正文有干货、结尾有引导关注
知乎风格要求：有深度、有数据、有"先问是不是再问为什么"的思辨
```

::: tip 实操记录
把这篇文章保存好，它将作为你模块3的产出物之一。
:::

## 第2课：AI设计入门 —— 一句话生成海报

**时长**: 约120分钟 | **类型**: 实操为主

### 核心概念

AI设计的本质是：**你当"需求方"和"审美判断者"，AI当"执行设计师"**。你不需要会画画，但需要知道什么是"好看"。

::: tip 非设计专业学生必读
你不是在和设计师竞争，而是在用AI补齐你不会设计的短板。一个人的团队，也能产出专业级的视觉内容。
:::

| 工具 | 最适合 | 免费程度 | 上手难度 |
|------|--------|---------|---------|
| **Canva**（canva.cn） | 海报、PPT、社交媒体图、简历排版 | 免费版功能丰富 | ★☆☆☆☆ 最低 |
| **即梦**（jimeng.jianying.com） | 中文图片生成、中国风、二次元 | 每天免费额度 | ★★☆☆☆ |

### 动手实操1：用AI生成一张活动海报

#### Step 1：用即梦/DALL·E 生成主视觉图（15分钟）

打开即梦（jimeng.jianying.com）或 ChatGPT（内置DALL·E），输入以下 Prompt：

```text
一张简洁现代的活动海报背景图，主题是"AI赋能未来教育"，
主视觉：一个发光的蓝色神经网络图案与一本打开的书本融合，
配色：深蓝底色 + 金色光效，
风格：科技感、简约、留白充足（适合在图片上加文字），
竖版比例3:4，高清
```

如果对生成结果不满意，尝试调整关键词：
- 想要更简约？加"极简主义、大量留白"
- 想要更温暖？把"深蓝+金色"改成"暖橙色+米白色"
- 想要更酷？加"赛博朋克风格、霓虹灯效果"

#### Step 2：在Canva中完成排版（20分钟）

1. 打开 [Canva](https://canva.cn)，注册免费账号
2. 点击"创建设计" → 选择"海报"（竖版）
3. 上传你刚才生成的AI图片作为背景
4. 添加文字：活动标题、时间、地点、二维码
5. 使用Canva的"魔术设计"（Magic Design）功能一键生成变体
6. 调整到你满意为止，导出PNG

#### Step 3：让AI帮你写海报文案（5分钟）

```text
我设计了一张活动海报，主题是"AI赋能未来教育"。
请帮我写海报上的文案，包括：
1. 主标题（6-10个字，有冲击力）
2. 副标题（15-20个字，补充说明）
3. 3个卖点短句（每个不超过8个字）
4. 一句号召行动的话（如"扫码报名"之类的）

风格：面向大学生，年轻化、有吸引力、不浮夸。
```

### 动手实操2：AI设计Prompt公式

**通用高质量Prompt公式**：

```
[主体] + [场景/背景] + [风格/媒介] + [光线/色调] + [构图/视角] + [画质参数]
```

一张简洁的扁平化插画风格图解，展示"人机协作"的概念，

阳光从窗户洒进来，柔和的暖色调，画面有"岁月静好"的感觉，

**个人Logo设计**：
```text

图形元素：一本书与一个发光的灯泡融合，
扁平化风格，渐变蓝色，白色背景，适合APP图标
```

**产品宣传图**：
```text
一张产品宣传图，展示一款手机App的界面，
手机悬浮在画面中央，周围是发光的粒子效果，
科技感，深色背景，电影级灯光，横向16:9
```

### 避坑指南

| ❌ 常见错误 | ✅ 正确做法 |
|-----------|-----------|
| "帮我做一张好看的海报" | 明确描述：主题、风格、配色、构图、用途 |
| 不满意就换一个工具 | 先调整Prompt，换个描述方式往往效果更好 |
| AI生成直接用，不检查 | AI经常把文字生成乱码，必须人工检查 |
| 只用AI生图，不会排版 | 用Canva排版，AI图片+人工排版=专业作品 |
| 追求一次到位 | 迭代3-5次，每次微调Prompt，直到满意 |

## 第3课：AI视频与音频制作 —— 从脚本到成片

**时长**: 约120分钟 | **类型**: 实操为主

### 核心概念

AI视频不是"AI一键生成完整视频"（目前技术还不够成熟），而是 **"AI辅助视频制作的每个环节"**。

**AI视频制作分工**：
- AI负责：脚本生成、配音、字幕、部分画面素材
- 你负责：选题、剪辑节奏、整体把控、最终审核

### AI视频工具速览

| 工具 | 最适合 | 核心能力 | 免费程度 |
|------|--------|---------|---------|
| **剪映**（桌面端） | 一站式视频制作 | AI文字成片、AI配音、AI字幕、AI数字人 | 基础免费 |
| **可灵**（kling.kuaishou.com） | AI视频素材生成 | 文生视频、图生视频 | 每天免费额度 |
| **即梦** | 图片/短视频素材 | 文生图、图生视频 | 每天免费额度 |
| **Runway** | 高级视频编辑 | 视频风格转换、运动笔刷、绿幕抠像 | 有免费试用 |

::: tip 本课推荐
**剪映 = 核心工具**。它是目前对中文用户最友好的AI视频工具，涵盖了从素材到成片的完整流程。
:::

### 动手实操：制作一个1分钟"我的专业介绍"短视频

#### Step 1：AI辅助选题与脚本（15分钟，推荐工具：DeepSeek）

```text
你是一个短视频创作专家。我需要制作一个1分钟的短视频，主题是"我的专业介绍"。

我的专业是：[你的专业名称]
目标受众：高中生和大学新生，想了解这个专业
视频风格：快节奏、有趣、有干货
时长：60秒

请帮我生成脚本：
1. 一个吸引人的标题
2. 按时间线分镜（5-8个分镜），每个分镜包含：
   - 画面描述（给AI生成素材用的）
   - 旁白文案（15-20字/分镜）
   - 时长分配
3. 视频的开头和结尾话术
```

#### Step 2：AI配音（10分钟，剪映内置）

1. 打开剪映，点击"开始创作"
2. 点击"音频" → "录音/配音" → 选择"文本朗读"
3. 粘贴你的旁白文案
4. 选择音色（推荐：阳光男生/甜美女声/新闻播报）
5. 调整语速（推荐1.1-1.2倍速，更适合短视频）
6. 试听并微调

#### Step 3：AI生成画面素材（20分钟，可灵/即梦 + 剪映）

**方案A（推荐）：剪映AI文字成片**
1. 剪映首页 → "AI文字成片"
2. 粘贴你的完整脚本文案
3. AI自动匹配素材、配音、字幕
4. 人工审核替换不合适的素材

**方案B：可灵生成自定义素材**
1. 将每个分镜的画面描述，输入可灵（kling.kuaishou.com）
2. 生成5秒视频片段
3. 下载后导入剪映

可灵Prompt示例：
```text
一个大学校园的航拍镜头，阳光明媚，学生们在草坪上学习和交流，
画面明亮温暖，电影感，写实风格，高清
```

#### Step 4：剪辑与字幕（15分钟，剪映）

1. 排列素材顺序，匹配配音节奏
2. 点击"文本" → "智能字幕" → 自动识别生成字幕
3. 检查字幕准确性（剪映识别率很高，但专业术语可能出错）
4. 添加背景音乐：剪映音乐库 → 选择"轻快/科技/校园"风格
5. 调整音乐音量（-20dB左右，不要盖过人声）
6. 添加片头片尾文字

#### Step 5：导出与发布（5分钟）

- 导出设置：1080p、30fps、竖屏9:16（适合抖音/小红书/视频号）
- 文件大小控制在100MB以内

## 第4课：AI数据分析 —— 不写代码也能做数据分析

**时长**: 约120分钟 | **类型**: 实操为主

### 核心概念

AI数据分析改变了游戏规则：**你不需要学Python或SQL，只需要上传数据文件，用自然语言告诉AI你想分析什么。**

::: danger 数据隐私红线
绝对不要上传包含个人隐私的数据（身份证号、手机号、家庭住址等）。上传前对敏感信息做脱敏处理（如将真实姓名替换为"学生A"）。
:::

### 支持AI数据分析的工具

| 工具 | 上传方式 | 分析能力 | 可视化 |
|------|---------|---------|--------|
| **ChatGPT** | 点击附件图标上传 | 强，支持复杂分析 | 可生成图表代码 |
| **Claude** | 拖拽文件到对话框 | 强，长文本优势 | 可生成图表代码 |
| **DeepSeek** | 点击附件上传 | 强，推理能力突出 | 可生成图表代码 |
| **Kimi** | 点击附件上传 | 强，适合多文件分析 | 可生成图表代码 |

### 黄金三步法

```
第1步：数据理解 → "请先帮我看看这个数据文件的基本情况"
第2步：分析探索 → "请帮我分析XX与XX之间的关系"
第3步：可视化建议 → "请帮我设计合适的图表来呈现这些发现"
```

### 动手实操：用AI分析一份数据

#### 准备工作

找一个你感兴趣的Excel或CSV文件。如果没有，可以自己做一个简单的：
- 本学期各科成绩
- 社团活动参与数据
- 日常消费记录
- 或者从网上找一个公开数据集

#### Step 1：数据概览与清洗（15分钟）

上传文件后，输入：

```text
我上传了一份数据文件。请先帮我了解这份数据：

1. 共有多少行、多少列？
2. 每列的字段名和数据类型是什么？
3. 哪些列存在缺失值？缺失比例是多少？
4. 数值列的基本统计描述（平均值、最大值、最小值、中位数）
5. 有没有明显的数据异常或可疑值？
```

#### Step 2：分析探索（20分钟）

以"社团活动参与数据"为例：

```text
基于以上数据，请帮我分析：

1. 哪种类型的活动参与率最高？哪种最低？
2. 参与人数是否随时间有明显变化趋势？
3. 工作日 vs 周末的活动参与率有差异吗？
4. 从数据中找出3个最有价值的洞察

请用非技术语言解释，就像在给不懂统计的同学做报告。
```

如果你是其他专业，可替换为：
- **经管**：分析销售数据的趋势、季节性、异常值
- **教育**：分析学生成绩的分布、影响因素、进步情况
- **医学**：分析患者数据中的相关性、风险因素

#### Step 3：可视化与报告生成（15分钟）

```text
请帮我做以下事情：

1. 推荐3种合适的图表类型来呈现以上分析结果
2. 说明每种图表应该展示什么数据
3. 给我在Excel/WPS中制作这些图表的具体步骤
4. 生成一份简洁的数据分析报告大纲（包含：核心发现、数据支撑、行动建议）

如果你能生成Python代码来绘制这些图表，也请一并提供，我可以复制到Google Colab中运行。
```

#### Step 4：用AI解读图表（10分钟）

如果你已经有图表（截图），可以上传给AI，让它帮你解读：

```text
我上传了一张数据图表的截图。请帮我：
1. 描述这张图表展示了什么信息
2. 指出3个最显著的趋势或异常点
3. 如果这是给老板汇报用的，你会怎么用一句话总结这张图？
```
