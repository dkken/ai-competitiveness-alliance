## 第1课：好Prompt vs 差Prompt —— 10倍效率差从哪来？

**时长**: 约120分钟 | **类型**: 认知 + 大量实操

### 核心概念

#### Prompt的本质

很多人以为跟AI对话就是"问问题"，这是最大的误解。Prompt的本质是：**给AI一个完整的上下文和工作指令**。

把AI想象成一个能力超强但完全不了解你处境的"新同事"：
- 你对它说"帮我写点东西" → 它需要猜你想要什么 → 差结果
- 你对它说"你是一位资深HR，帮我优化这份简历..." → 它明确知道角色和任务 → 好结果

#### Prompt的四个核心要素

| 要素 | 说明 | 示例 |
|------|------|------|
| **角色（Role）** | 设定AI的专业身份或视角 | "你是一位有10年经验的资深HR" |
| **任务（Task）** | 明确要完成的具体工作 | "请帮我优化这份简历" |
| **格式（Format）** | 指定输出的结构和样式 | "用表格形式，列出修改前后对比" |
| **约束（Constraint）** | 设定边界和限制条件 | "保持原意不变，控制在500字以内" |

### 动手实操：同一个任务，不同Prompt的差距

#### 任务：写一份求职意向说明

**差Prompt（V1）**：
```
帮我写一段求职意向
```

**中庸Prompt（V2）**：
```
帮我写一段求职意向，我是计算机专业的大四学生
```

**好Prompt（V3）**：
```text
你是一位资深职业规划顾问。请帮我写一段求职意向（100-150字），用于简历顶部。

我的背景：
- 专业：市场营销，大四在读
- 实习经历：在某快消公司做过3个月的市场部实习生
- 目标岗位：互联网公司的用户运营或市场推广岗位
- 个人优势：数据分析能力较强（会Excel和SQL），善于用数据驱动决策

要求：
1. 突出"数据分析+市场营销"的复合能力
2. 语气自信但不浮夸
3. 适合互联网行业
```

#### 对比实测

现在，请你打开任意一个AI工具，分别用上面V1、V2、V3三个Prompt测试，对比输出结果并记录：

| 对比维度 | V1（一句话） | V2（加了背景） | V3（完整指令） |
|---------|------------|--------------|--------------|
| 输出长度 | | | |
| 具体性（是否有空话） | | | |
| 可用性（能直接用在简历上吗） | | | |
| 个性化程度 | | | |

::: details 参考分析
你会发现：
- **V1**：输出的是AI编造的"万能模板"，跟你的真实情况毫无关系
- **V2**：好了一些，但仍然很泛，"计算机专业"这个信息太单薄
- **V3**：AI能够基于你的真实经历，写出有针对性的、可直接使用的求职意向

**关键认知**：不是AI不行，而是你没给它足够的信息。
:::

### 避坑指南：最常见的5个Prompt错误

| 错误类型 | ❌ 错误写法 | ✅ 正确写法 |
|---------|-----------|-----------|
| **太模糊** | "帮我写点东西" | "帮我写一篇800字的求职信，申请字节跳动的产品经理实习岗" |
| **缺少角色** | "分析一下新能源汽车市场" | "你是麦肯锡的高级分析师，请分析2025年中国新能源汽车市场的竞争格局" |
| **没有格式** | "给我一些建议" | "请用表格形式列出5个建议，每个包含：建议内容、实施难度（高/中/低）、预期效果" |
| **一次问太多** | "帮我写论文、做PPT、分析数据..." | 一次只做一件事，复杂任务分步完成 |
| **不接受修改** | "就这样吧" | "请从3个不同角度重新分析"、"写太长了，精简到200字"、"换一种更口语化的风格" |

## 第2课：六大Prompt框架详解

**时长**: 约120分钟 | **类型**: 认知 + 实操

### 框架1：角色扮演法（Role-Playing）

**原理**：给AI指定一个专业角色，让它在特定领域知识框架内回答。

**模板**：
```text
你是一位[角色名称]，拥有[X年经验/专业背景]。
请以你的专业视角，完成以下任务：[任务描述]
```

**实战示例**：

```text
你是一位有15年经验的大学招生面试官，面试过5000+名申请者。
我是一名高中生，正在准备大学自主招生面试。
请模拟面试场景，连续问我5个问题，并在我回答后给出反馈和改进建议。
```

**适用场景**：求职面试模拟、专业咨询、创意策划、困难对话排练

### 框架2：分步指令法（Step-by-Step）

**原理**：把复杂任务拆解为清晰的步骤，每一步AI都只需专注一个小目标。

**模板**：
```text
请按照以下步骤完成任务：

第一步：[步骤1内容]
第二步：[步骤2内容]
第三步：[步骤3内容]
...
```

**实战示例**：

```text
请按以下步骤帮我做一份"大学生暑期义工旅行"的方案：

第一步：列出5个国内适合大学生暑期义工旅行的目的地，每个附一句话说明推荐理由
第二步：选择一个你觉得最适合大学生第一次义工旅行的目的地，详细说明理由
第三步：为这个目的地制定7天的行程安排（含交通、住宿、义工内容、自由活动）
第四步：做一个预算表，估算整个行程的费用
```

**适用场景**：项目规划、方案设计、复杂分析

### 框架3：思维链法（Chain of Thought）

**原理**：要求AI"一步一步思考"，展示推理过程而非只给结论。

**模板**：
```text
请一步一步思考这个问题：[问题]
在给出最终答案之前，请先展示你的分析过程。
```

**实战示例**：

```text
请一步一步思考这个问题：我是一名大二学生，专业是环境工程，GPA 3.4。
我数学不太好但文字能力不错，对环保行业有兴趣。
请分析我应该走"考公"、"考研"还是"直接就业"三条路，并给出建议。

在给出建议之前，请先：
1. 分析每条路的环境工程专业适配度
2. 结合我的个人优劣势
3. 分析2026年环保行业的就业趋势
4. 然后综合给出建议
```

**适用场景**：决策分析、复杂推理、方案评估

### 框架4：示例引导法（Few-Shot Learning）

**原理**：给AI看几个你想要的"样例"，让它模仿输出格式和风格。

**模板**：
```text
请参考以下格式和风格完成任务：

示例1：
输入：XXX
输出：XXX

示例2：
输入：XXX
输出：XXX

现在请处理：[你的输入]
```

**实战示例**：

```text
请参考以下小红书爆款文案的格式，帮我写一篇推荐《原子习惯》这本书的笔记：

示例：
📚 一周读完一本书 | Day 1
《被讨厌的勇气》
⭐️⭐️⭐️⭐️⭐️

💡 一句话总结：
你的烦恼，都来自人际关系。

📝 三个让我醍醐灌顶的观点：
1. 课题分离：别人怎么看你，是别人的课题
2. 目的论：不是过去的经历决定现在的你
3. 共同体感觉：幸福来自对他人的贡献

🎯 适合谁读：
- 被社交焦虑困扰的人
- 总在意别人看法的人

---
现在请用同样的格式，帮我写《原子习惯》的推荐笔记。
```

**适用场景**：内容创作、风格模仿、格式化输出

### 框架5：反向提问法（Reverse Questioning）

**原理**：让AI在回答之前先问你澄清问题，确保它真正理解了你的需求。

**模板**：
```text
在回答我之前，请先问我3-5个澄清问题，以确保你完全理解我的需求。
等我回答了这些问题后，你再给出最终方案。
```

**实战示例**：

```text
我需要做一个关于"大学生心理健康"的调研报告。
在开始之前，请先问我5个关键问题，帮助我理清调研思路。
等你了解了我的具体需求后，再帮我设计调研方案。
```

**适用场景**：需求模糊时、复杂项目启动、方案设计

### 框架6：迭代优化法（Iterative Refinement）

**原理**：不要指望一次就能得到完美输出。通过多轮对话逐步打磨。

**经典迭代流程**：
```text
第1轮：给出初版指令 → AI给出初稿
第2轮："太长了，精简到300字" 或 "请从3个不同的角度重新分析"
第3轮："第2个角度最好，请重点展开" 或 "语气太正式了，换成口语化风格"
第4轮："很好，再加一个案例来说明第3点"
第N轮：直到满意为止
```

**实战示例**：

```text
（第1轮）
请帮我写一段产品宣传文案，产品是一款面向大学生的AI学习助手App。

（AI输出...）

（第2轮）
写得太像广告了，大学生会反感。换成"朋友推荐"的口吻，真诚一点。

（AI输出...）

（第3轮）
好多了！但是感觉有点啰嗦。把核心卖点压缩到3个，每个最多一句话。

（AI输出...）

（第4轮）
完美。在文案最后加一句引导下载的话。
```

### 综合练习：选3个框架实战

从以下任务中任选3个，各用1种框架完成：

1. 用**角色扮演法**，让AI模拟一位面试官，帮你练习自我介绍
2. 用**分步指令法**，让AI帮你做一个完整的春游/团建活动方案
3. 用**思维链法**，分析你适合考研还是直接工作
4. 用**示例引导法**，让AI模仿某篇爆款文章的标题风格生成5个标题
5. 用**反向提问法**，让AI帮你梳理"寒假应该做什么"的思路
6. 用**迭代优化法**，写一段个人简介，至少经过3轮优化

## 第3课：场景化Prompt实战

**时长**: 约120分钟 | **类型**: 实操为主

### 场景一：学业场景

#### 论文摘要生成
```text
你是一位学术写作导师。我的论文标题是《社交媒体使用强度对大学生学业成绩的影响——以XX大学为例》。
请帮我写一段200字左右的论文摘要，结构如下：
1. 研究背景与目的（1-2句）
2. 研究方法（1句）
3. 主要发现（2-3句）
4. 结论与启示（1-2句）
```

#### 文献综述梳理
```text
我最近读了5篇关于"AI教育"的论文，请帮我整理一份文献综述框架。
论文主题分别是：
1. AI在个性化学习中的应用
2. 大模型对高等教育的冲击
3. AI辅助写作的教学效果
4. 学生使用AI的学术诚信问题
5. AI时代教师角色的转变
请用表格形式梳理，每篇论文列明：核心观点、研究方法、与我研究方向的关联。
```

#### 知识梳理与复习
```text
你是一位大学老师。我正在复习《宏观经济学》期末考，请帮我梳理以下3个章节的核心知识点：

第1章：GDP核算
第2章：通货膨胀与失业
第3章：货币政策与财政政策

要求：
1. 每个章节列出5个最重要的概念，每个概念用一句话解释
2. 做一个对比表格，对比货币政策与财政政策的异同
3. 给出5道模拟考题（选择题），并附答案
```

### 场景二：求职场景

#### 简历优化
```text
你是一位在500强企业有10年招聘经验的HR总监。
请帮我优化以下简历中的"实习经历"部分，使它更符合互联网大厂产品经理实习岗的要求：

[在这里粘贴你的实习经历原文]

优化要求：
1. 用STAR法则（情境-任务-行动-结果）重写每段经历
2. 突出可量化的成果（如：提升了XX%的效率、获得了XX个用户）
3. 用积极的动作词开头（如：主导、推动、设计、优化）
4. 控制总长度在5条以内，每条不超过2行
```

#### 面试模拟
```text
你是一位来自字节跳动的高级产品经理。现在你正在面试一位申请产品经理实习岗的大学生。

请进行一场模拟面试：
1. 先问我5道行为面试题（如："请讲一个你在团队中解决冲突的经历"）
2. 然后给我3道产品案例分析题
3. 最后让我提问你3个问题

每题我回答后，请给出1-10分评分和30字以内的改进建议。
```

#### 行业研究报告
```text
请帮我做一份"2026年AI教育行业"的简要分析报告：
1. 市场概况：AI教育市场规模和增长趋势
2. 主要玩家：列出5家代表性公司的核心产品
3. 趋势分析：3个最重要的行业趋势
4. 机会分析：作为求职者，哪些岗位在增长
要求：每个部分200-300字，用事实和数据说话。
```

## 第4课：AI协作高级技巧

**时长**: 约120分钟 | **类型**: 认知 + 实操

### 技巧1：上下文管理

AI的记忆长度有限（即使是长上下文模型，信息密度也会下降）。学会管理上下文，是让AI持续高效协作的关键。

#### 黄金法则

1. **重要信息放在对话开头**：AI对开头和结尾的信息关注度最高
2. **定期"总结重置"**：对话过长时，让AI总结要点，然后在新对话中继续
3. **分场景开新对话**：不同任务用不同的对话窗口

#### 实操：用AI帮你管理上下文

```text
我们正在讨论一个复杂的项目方案。我已经给你提供了大量背景信息。
现在请帮我总结一下：
1. 我们目前达成的共识（3条）
2. 尚未解决的问题（列出所有）
3. 下一步我需要做的决策（按优先级排列）

然后，我会在新的对话中继续这个话题。
```

### 技巧2：多轮对话策略 —— "从宽到窄"

**漏斗模型**：先宽泛探索，再逐步聚焦。

```text
第1轮（发散）：列出XX领域的10个可能方向
第2轮（筛选）：从这10个中，选出你觉得最有前景的3个，并说明理由
第3轮（深入）：针对第1个方向，做一个详细的分析报告
第4轮（优化）：这个分析还不够具体，加入数据支撑和竞品对比
第5轮（应用）：基于以上分析，给我一个可执行的行动计划
```

**实战练习**：选一个你感兴趣的领域（比如"AI+教育"、"AI+医疗"、"AI+电商"），用漏斗模型完成一轮从发散到聚焦的探索。

### 技巧3：给AI"喂"资料

AI能读取你上传的文件。利用这个能力，让它基于真实资料工作。

**支持的文件类型**：PDF、Word、Excel、PPT、图片、代码文件

**最佳实践**：
```text
我上传了一份[文件类型]：[文件简述]。
请仔细阅读这份文件，然后完成以下任务：
1. 用3句话总结这份文件的核心内容
2. 列出文件中提到的所有数据（附上来源页码）
3. 找出文件中三个最有价值的洞察
```

### 技巧4：多AI协同

不同AI有不同特长。学会"取长补短"：

| 任务环节 | 推荐工具 | 原因 |
|---------|---------|------|
| 深度调研、长文档分析 | Kimi | 200万字上下文，精读能力最强 |
| 逻辑推理、复杂分析 | DeepSeek | 推理能力国产最强，数学和编程优秀 |
| 创意文案、中文创作 | 豆包 | 中文网感最好，响应最快 |
| 英文写作、专业报告 | Claude | 长文本处理强，输出质量高 |
| 综合对话、多语言 | ChatGPT | 综合能力最均衡 |

**组合工作流示例**：

```text
任务：写一篇关于"AI对大学生就业影响"的深度文章

Step 1（Kimi）：上传5篇相关研究报告，让Kimi提炼核心数据和观点
Step 2（DeepSeek）：输入Kimi的提炼结果，让DeepSeek做逻辑梳理和分析框架
Step 3（Claude）：输入分析框架，让Claude生成初稿
Step 4（豆包）：用豆包润色中文表达，调整语感和风格
```

这是你学完本模块后最重要的"资产"。建议用飞书或腾讯文档建立一个模板库：

**模板库结构建议**：

```

│   ├── 考试复习
│   ├── 文献整理

每个模板格式：

```

**目标**：本模块结束时，你的模板库里至少有20个Prompt模板。
