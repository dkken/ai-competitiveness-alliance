## 阶段概览

第一阶段是课程的基石。无论你选择就业轨还是创业轨，都需要先掌握AI的基本认知和通用能力。四个模块环环相扣，从认知破壁到工具精通，再到专业结合，形成完整的AI能力基础。

```
模块1（第1周）→ 模块2（第2周）→ 模块3（第3周）→ 模块4（第4周）
  认知破壁           Prompt工程          工具深度实战         AI+X专业结合
```

| 模块 | 标题 | 核心能力 | 课时 | 产出物 |
|------|------|---------|------|--------|

## 为什么第一阶段是必修的？

### 没有地基，盖不了高楼

很多同学会问："我能不能直接学就业课或创业课？"答案是：**不能。** 原因很简单：

- **模块1** 解决"AI是什么"的问题——如果你连AI能做什么、不能做什么都不知道，后面所有内容都无从谈起
- **模块2** 解决"怎么和AI沟通"的问题——Prompt是AI时代的基础语言，就像打字是计算机时代的基础技能
- **模块3** 解决"AI能帮我做什么"的问题——让你亲身体验AI在写作、设计、视频、数据分析等场景的实际效果
- **模块4** 解决"AI和我的专业怎么结合"的问题——这是你差异化竞争力的核心，也是后续选择就业轨还是创业轨的关键依据

## 第一阶段的学习成果

完成第一阶段后，你将拥有：

1. **AI认知体系**：清楚知道AI能做什么、不能做什么，什么场景下用哪个AI工具
2. **Prompt工程能力**：掌握6大Prompt框架，拥有20+场景的Prompt模板库
3. **AI工具实战经验**：亲自动手完成过AI写作、AI设计、AI视频、AI数据分析
4. **AI+X定位报告**：明确你的专业与AI的最佳结合点，知道后续该走就业轨还是创业轨

## 学习建议

::: tip 每周投入时间
约 8-10 小时（含实操）。建议每天安排1-2小时，不要集中在周末突击，AI工具的使用需要"手感"。
:::

::: warning 学习顺序
请按模块顺序学习，每个模块的内容都建立在之前模块的基础上。跳模块学习会导致后续内容理解困难。
:::

::: info 作业提交
每个模块的作业通过腾讯文档提交，具体链接见各模块末尾。作业是解锁下一模块的前置条件。
:::

::: danger 常见误区
- ❌ "我有编程基础，可以跳过模块1" → 模块1不是教编程，而是教AI认知和工具使用，CS学生也需要
- ❌ "我会用ChatGPT，Prompt不用学" → 大部分人的Prompt水平停留在"随便问"，模块2教你的是"精准协作"
- ❌ "模块3的AI视频和设计跟我的专业无关" → 这些是"能力展示"工具，任何专业都需要
:::
